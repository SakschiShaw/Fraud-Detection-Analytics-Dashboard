# Fraud Detection Dashboard (SQL + Power BI)

## Project Overview

Developed a fraud detection analytics dashboard using SQL and Power BI to identify suspicious transaction patterns across 500 financial transactions.

## Key Insights

* Identified  fraud rate across dataset
* High-value transactions (> $500) showed **higher fraud probability**
* Evening transactions showed increased fraud activity
* Detected **high-risk transactions using behavioral risk scoring**

## Tools Used

SQL
Power BI
Excel

* Dataset (CSV)

## Business Value

The dashboard helps financial teams detect fraud patterns early and monitor suspicious transaction behavior in real time.
